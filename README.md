# Magisk Camera2 API Enabler

---
# **Module Name**
## Description
This Magisk module enables the new Camera2 API by adding "persist.camera.HAL3.enabled=1" to build.prop.
## Changelog
update magisk template to v15.0
## Requirements
- Magisk v15.0

- Flash this zip
